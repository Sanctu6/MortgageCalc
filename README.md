# Іпотечний Калькулятор

Сучасний веб-застосунок для розрахунку іпотечних платежів та графіків амортизації. Спеціалізований для українського ринку з підтримкою програм єОселя та стандартних комерційних іпотек.

## Функціональність

- **Два режими розрахунку:**
  - По термінам: розрахунок щомісячного платежу за вказаний період
  - По платежу: розрахунок строку погашення при комфортному платежі

- **Мультивалютність:**
  - Підтримка UAH та USD
  - Автоматичне конвертування при зміні валюти
  - Налаштовуваний курс обміну

- **Додаткові витрати:**
  - Пенсійний фонд (1% від вартості)
  - Страховка (% від залишку кредиту)
  - Разові комісії (оцінка, нотаріус, банк)

- **Детальні результати:**
  - Щомісячний платіж з урахуванням страховки
  - Реальний термін погашення
  - Загальна переплата
  - Повний графік амортизації з розбиттям на основне тіло, відсотки та страховку

## Архітектура

Код організований як IIFE (Immediately Invoked Function Expression) модуль з наступною структурою:

- **Constants**: Налаштування та константи застосунку
- **State Management**: Локальний стан для отримання значень між операціями
- **DOM Caching**: Кеш посилань на DOM елементи для продуктивності
- **Event Listeners**: Централізована реєстрація обробників подій
- **Utility Functions**: Допоміжні функції (форматування, конвертація)
- **Calculation Logic**: Основна бізнес-логіка розрахунків

## Особливості

- **Валідація**: Перевірка коректності введених даних
- **Форматування**: Локалізована валюта в українській та англійській мовах
- **Цифрова точність**: Використання EPSILON для уникнення помилок при роботі з float
- **Производительность**: DOM кешування та ефективна обробка подій
- **Зручний UX**: Автоматичний розрахунок при зміні будь-якого параметра

## Формули

### Ануїтетний платіж (Режим по термінам)

```
A = P × (r × (1+r)^n) / ((1+r)^n - 1)
```

Де:
- A = щомісячний платіж
- P = залишок кредиту
- r = місячна ставка (річна ставка / 12 / 100)
- n = кількість місяців

### Страховка

```
Insurance = (Balance × Annual Rate) / 12
```
